## Problem

Given an integer array nums and an integer k, return the k most frequent elements. 
You may return the answer in any order.

```
Input: nums = [1,1,1,2,2,3], k = 2
Output: [1,2]
```

```
Input: nums = [1,2,1,2,1,2,3,1,3,2], k = 2
Output: [1,2]
```

## Optimal approach
The optimal approach includes storing the frequency of all the elements in a hashmap, then utilizing the
bucket sort algorithm to sort the elements into an array as per their frequency. From these, the k most
frequent can then be chosen.


## Solution

```java
class Solution {
    public int[] topKFrequent(int[] nums, int k) {
        HashMap<Integer, Integer> freq = new HashMap<>();
        for (int num : nums) {
            freq.put(num, freq.getOrDefault(num, 0) + 1);
        }
        List<Integer>[] buckets = new List[nums.length + 1];
        for (int i = 0; i < buckets.length; i++) {
            buckets[i] = new ArrayList<>();
        }
        for (int num : freq.keySet()) {
            buckets[freq.get(num)].add(num);
        }
        int[] res = new int[k];
        int idx = 0;
        for (int i = buckets.length - 1; i >= 0 && idx < k; i--) {
            for (int num : buckets[i]) {
                res[idx++] = num;
                if (idx == k) return res;
            }
        }  
        return res;
    }
}
```
